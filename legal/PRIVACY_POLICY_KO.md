# 개인정보처리방침

**최종 수정일: 2026년 1월 19일**

본 개인정보처리방침은 Flan("본 앱")이 사용자의 개인정보를 어떻게 처리하는지 설명합니다.

## 1. 개인정보의 수집 및 저장

본 앱은 **서버를 운영하지 않으며, 사용자의 개인정보를 수집하거나 저장하지 않습니다**.

모든 데이터(채팅 내역, 캐릭터 설정, 프롬프트 설정, API 키 등)는 사용자의 기기에만 저장되며, 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

## 2. 제3자 서비스 이용

본 앱은 사용자가 입력한 API 키를 사용하여 Google Gemini API와 직접 통신합니다.

- 본 앱은 사용자와 API 간의 통신을 중계만 할 뿐, 데이터를 저장하거나 수집하지 않습니다.
- 사용자의 API 키와 메시지는 사용자의 기기에서 직접 Google로 전송됩니다.
- API 사용과 관련된 데이터 처리는 [Google AI 서비스 약관](https://policies.google.com/terms/generative-ai)에 따릅니다.

## 3. 사용자의 권리

사용자는 언제든지 앱 내에서 자신의 데이터를 확인하거나 삭제할 수 있으며, 앱을 삭제하여 모든 데이터를 완전히 제거할 수 있습니다.
